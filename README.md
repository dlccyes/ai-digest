# My Personal Blog

Getting started

```
git clone https://github.com/dlccyes/blog
cd blog
make init
```

And then use Obsidian, VsCode, or another good markdown editor to open the directory.

To see available commands

```
make help
```

## Creating a post with local images

Create a new directory and put your post inside, name it `index.md`. Put the images you want to reference in the same directory.

Please ensure that the size of each new image is < 1MB.
